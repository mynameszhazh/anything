# 核心模块

## core(核心)
- cesuim 的最底层的模块, 不依赖其它库, 主要提供一些基本的工具函数和数据结构, 例如数学运算、事件处理、时间处理等.

## scene(场景)
- cesuim 的场景模块, 依赖core模块,负责渲染3d世界, 直接和webgl交互, 
- 主要组成部分有:
  - 渲染引擎: 相机控制, 着色器
  - 几何体系统: 图元, 纹理, 材质
  - 地形
  - 图层
  - 特效

## dataSouces(数据源)

- 高级api层, 用于简化数据集成和处理, 管理实体和数据

## wedgets(控件)

- 最上层的用户界面组件, 提供用户交互和视觉控制元素
