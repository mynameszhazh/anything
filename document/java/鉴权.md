# 用户鉴权

## cookie
> 他会自动带上 cookie进行操作

### 优点
- http原生支持的很好

### 缺点
- 不能进行跨域操作
- app中不能使用
- 用户可以自动更改cookie
- 存储小
- 不安全

## session
> 注意 前端还是使用cookie进行操作

### 优点
- 安全

### 缺点
- 不同服务中 不能共享
- 包含所有的cookie问题


## token
> 令牌技术

### 优点
- 支持PC端, 移动端
- 解决了集群环境下的认证问题
- 减轻服务器端存储压力

### 缺点
- 实现起来,会多很多的步骤
  - 看起来跟没有一样

### JWT
- 库的语法进行哦相关的操作
