### 父子报表

> 复杂报表或数据内容较多的内容，可以将报表分成多个子报表，然后将子报表嵌入到主报表中，这样可以使报表的结构更加清晰，也方便报表的维护。

- 一个主报表可以包含多个子报表，子报表也可以包含子报表，这样就形成了一个报表的树形结构。
- 子报表数据 也是通过 params 传递 里面需要存在 sublist subpath 这些参数
